# Frontend PWA

Folder ini adalah fondasi frontend baru untuk `GitHub Pages + PWA`.

Tujuan tahap awal:

- memisahkan frontend modern dari UI Apps Script lama
- menyiapkan app shell mobile-first
- menyiapkan manifest dan service worker
- mempertahankan Apps Script sebagai backend sementara

Tahap berikut yang direncanakan:

1. sambungkan frontend ini ke endpoint Apps Script
2. buat role-based screen untuk publik, bendahara, dan pengurus
3. tambahkan login flow yang terasa responsif
4. deploy ke GitHub Pages
